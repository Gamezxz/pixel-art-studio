Or clean AI-generated art: the hybrid pipeline

Point pixelpipe at any image model's output (or a screenshot). It recovers the true pixel grid, strips baked-in backgrounds, hardens alpha, kills orphan noise, collapses the color explosion, and locks everything onto one shared palette — then hands you a build script to keep editing and animating. Every asset shares one color vocabulary, so a whole cast looks like one game.

a real run: codex-generated chibi cast (196,703 colors) → cleaned, background-stripped & palette-locked (31 shared colors, ~64px each) by pixelpipe in one pass

python3 scripts/pixelpipe.py generated.png \
  --strip-checker --max-colors 24 --display 6
# or lock every asset to one game palette:
python3 scripts/pixelpipe.py generated.png --palette my_game

…then animate it deterministically — the same sprite, every frame:

A codex-generated knight (174,172 colors) → pixelpipe dominant-block sampling (128px · 28 colors). Idle is puppeted from the single cleaned sprite; the walk is a codex-generated 6-frame walk-cycle sheet cleaned in one pass, so every frame shares one palette. No re-rolling between frames — the build script is the animation.

What's inside

✍️

pixelstudio.py

Frames × layers engine: pixel-perfect primitives, palette locking, hue-shifted ramps, Bayer dithering, selective outlines, masked painting via only=.

🔍

study.py

Learns from reference art: recovers true resolution from sloppy upscales, strips baked-in checkerboards, extracts palettes, ramps and technique stats.

📚

Knowledge base

Proportions, silhouette-first workflow, shading & color theory, animation timing tables, retro palettes (Game Boy, PICO-8, C64…), engine export guides.

🧠

It grows

Study an image → Claude writes an imperative style card → those rules and palettes apply to every future artwork. The chibi cast follows one learned card studied from reference art.

🎞️

Real deliverables

1x PNG masters, animated GIFs with proper transparency, spritesheets with Aseprite-compatible JSON for Unity / Godot / Phaser.

♻️

Deterministic

The build script is the artwork — diffable, re-renderable, animation frames are function calls, not hand-copied pixels.

Ten lines to a shaded sprite

from pixelstudio import Sprite, ramp

BODY = ramp("#38b764", 5)                 # hue-shifted 5-step ramp
s = Sprite(32, 32, palette=BODY + ["#1a1c2c"])

s.ellipse(6, 12, 25, 27, BODY[0])                # darkest = silhouette
s.ellipse(5, 11, 23, 26, BODY[2], only="opaque") # restack lighter, toward light
s.outline(BODY[0], where="inside")              # selective outline

s.preview("preview.png", scale=10)   # Claude LOOKS at this
s.save_gif("out.gif", scale=6)
